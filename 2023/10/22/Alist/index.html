Alist

Alist 介绍

！！！可以直接播放网盘里的视频，原画输出，不用会员

Alist 是一个文件列表程序，它支持多种存储，支持网页浏览和 WebDAV, 它也是跨平台程序，由 gin 和 Solidjs 支持。 用 Alist 管理多个网盘非常方便。 以下是个人的经验，仅供参考。友善的建议欢迎提出,有好的经验也欢迎分享。

效果展示

下载安装

下载地址：Releases · alist-org/alist · GitHub

以Windows为例，下载指定版本的文件。

解压文件并进入文件夹；运行CMD，并输入.\alist.exe server，运行程序；

注意看start server@0.0.0.0:5244 的输出，之后没有报错，说明操作成功，最后也可以看到初始密码，用户名默认是admin：

在浏览器中输入本地地址，加alist端口号，即可看到登陆页面了：

用户名默认是admin，输入用户名和刚才黑窗口的密码，就可以登陆成功了：

在管理里面，可以更改管理员信息，用户名和密码都可以改：

添加网盘

不同网盘添加方法有差异，有的存在时效性，有的永久登录，
此处列出四种网盘添加方法，其它的请参考 Alist 官方手册。
添加存储网盘时，那些通用选项点击此处查看介绍

天翼云盘

  1. 给天翼云盘账号设置一个密码。步骤，手机天翼云盘 APP,
    依次点击右下角“我的”，左上角“账号与安全”，滚屏到底部“安全中心”，
    “账号安全管理”，“密码强度”，在此处可以设定或修改密码。
  2. 电脑浏览器进入“alist控制台”，并且登录用户，
    依次点击管理，存储，添加按钮，可能需要展开菜单才能找到“存储”。
  3. 选择并填写重要信息。
    驱动 组合框，选择天翼云盘客户端，不是天翼云盘。
    挂载路径 编辑框，填写一个自定义的名称，斜杠符号开头，例如 /tyyp
    如果只是填写一个斜杠，则会把网盘主目录列出在首页，可能会混淆其它网盘。
    WebDav 策略 组合框，选择本地代理，最佳兼容性。
    提取文件夹 组合框，让文件夹排序在前，还是排序在文件末尾。
    用户名 编辑框，天翼云盘账号，一般是手机号。
    密码 编辑框，天翼云盘密码。
    排序 组合框,按需选择，名称、大小、日期。
    排序方式 组合框，按需选择，升序、降序。
    类型 组合框，选择个人云。
    其它未提到的选项保持默认，无需更改。
    点击“添加”按钮即可完成，回到 Alist 主页就能浏览和管理天翼云盘文件。

123云盘

  1. 要有个账号和密码， 点此进入 123云盘官网
    选择微信登录，扫码确定后，绑定手机号，然后设置个密码。
  2. 在 Alist 管理后台添加存储，方式和添加天翼云盘相同。
  3. 添加时填写重要信息，
    驱动 组合框，选择 123云盘，
    其余选项和天翼云盘的类似，
    用户名和密码填写 123云盘的账号和密码，
    根文件夹ID 编辑框，保持默认，无需更改。
    最后点击“添加”按钮完成。

百度网盘

  1. 先 点击此处进入网页
    进入后找到:“点击这里open in new window 来获取刷新令牌。”

  2. 点击那链接后，根据页面提示登录百度账号。
    登录成功后，根据页面提示确认受权。

  3. 确认后页面会出来一堆英文，类似这样:

    把网页上这段内容复制下来留作备用。

  4. 进入 Alist 的后台管理，添加存储，方式参考上文天翼云盘部分。

  5. 添加时填写重要信息，部分内容参考上文天翼云盘。
    驱动 组合框，选择百度网盘。
    选中 web代理 复选框。
    WebDav策略 组合框，选择 本地代理。
    刷新令牌 编辑框，填写之前复制的， refresh_token 后面的那一串数字和字母。
    下载接口 组合框，选择官方，很稳定，速度慢，
    由于百度网盘 API 的限制，下载大于 20M 左右的文件需要携带 header：“User-Agent”
    详情查看 Alist 官方用户手册，或者使用本程序中的代理功能进行传输。
    下载接口 选择非官方，不稳定，不用配置 UserAgent.
    客户端ID 编辑框，和客户端秘钥 编辑框，默认已经填写好，
    如若不然请从之前复制的内容中找寻。 client_id 后面的一串字符就是客户端ID，
    client_secret 后面的一串字符就是客户端秘钥。

  6. 填写完成后点击“添加”按钮即可。

阿里云盘

  1. 首先需要获取 refreshToken 刷新令牌，可点此参考
    方法一：从安卓手机提取，很安全，不经过第三方。
    先安装 阿里云盘APP 并登陆，后安装 MT管理器 或者别的工具能访问“手机存储/android/data/”文件夹也可。
    可点此从酷安下载 MT管理器
    可点此从官方下载 cx文件浏览器
    可点此从 softpedia 下载 TotalCommander安卓版
    点此可下载 x-plore文件管理器
    使用 MT管理器 进入手机存储的这个文件夹“/Android/data/com.alicloud.databox/files/logs/”
    如果以前登陆过 阿里云盘APP, 那么先删除“logs”目录下的“trace”文件夹，再重新打开 阿里云盘APP 若提示登陆就登陆账号。
    完成后重新打开刚才的“logs”目录，进入“/trace/xxxx/yunpan/”找到“日期命名的.log”文件。
    （其中“xxx”是一串数字，或者是字母数字混合，每个人的都不一样。）
    之后可以直接在手机上打开“*.log”文件，或者把它发送到电脑用记事本打开也可以。
    接着在“*.log”文件内搜索“refreshToken”，在它后面紧跟着的一串字母数字就是刷新令牌。
    方法二：先点击此处进入网页
    然后使用阿里云盘APP 扫码确认登录，稍等几秒钟，
    在页面内的编辑框里就解析出来了 refresh_token, 点击 copy按钮 就复制到了剪贴板。
    或者 点击此处进入这个网页
    找到“点击:获取 Token”按钮，点击后扫码登录，
    按页面提示，稍等再点击，也能解析出 refresh_token.
  2. 进入“alist控制台”，管理，存储，添加，方法参考上文天翼云盘部分。
  3. 填写关键信息，部分内容参考上文天翼云盘。
    驱动 组合框，选择阿里云盘，不是阿里云盘分享。
    根文件夹ID 编辑框，保持默认，无需更改。
    刷新令牌 编辑框，填写之前获取的那一串字符。
    秒传 复选框，按需选择。
    完成后点击“添加”按钮即可。

补充说明

  1. 本压缩包内的“StopAlist.cmd”，点击会停止 Alist程序。
  2. “运行alist服务.cmd”，点击会打开命令行窗口，而不是后台运行。
  3. 软件不定期更新，有需要者可从官方下载。

常见问题

Q. 如何让 Alist 开机自启动？
A. 定位到“alist.exe”所在目录，找到“HideRunAlist.vbs”，
右键菜单创建快捷方式,然后将该快捷方式移动到这个文件夹内即可
“C:\ProgramData\Microsoft\Windows\Start Menu\Programs\Startup”，
或者这个“C:\Users\你的用户名\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup”。
Q. 如何让 Alist 停止记录日志？
