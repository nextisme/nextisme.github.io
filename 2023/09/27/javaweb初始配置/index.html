javaweb初始配置

发表于2023-09-27|更新于2023-09-27|Knowledge

Eclispse开发web项目

1.关联tomcat

左上角window - preferences - server - Runtime Environments - add

配置 tomcat 和 JRE

2.在下方面板click server

点击蓝色的字即可

3.新建项目

左上FIle - new - other - Dynamic Web Project(动态网页)

选择tomcat的版本

配置web版本是 2.5

最后打勾选项

4.启动

右下Server - Add and Remove - 选中项目名 然后Add

右键 srart

5.配置tomcat运行环境

在主项目右键 Build Path - Libraries - Add Librarie - Server Runtime - Next - tomcat

6.统一编码

charseet pageEncoding

换三个UTF-8

或

全统一设置 左上角window - preferences - JSP Files - Encoding - iso utf-8

7.其他

在web.xml可以配置启动顺序

html注释

java注释 // /* */

jsp注释 <%– –%>
